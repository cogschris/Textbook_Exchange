About the Project

This site is designed to be a marketplace for USC students to buy and sell their textbooks. Users will be able to upload textsbooks they want to sell, and users can search for said textbooks by name, class, or subject.

Any user can search for textbooks and reach out to the original owners if they are interested in buying. However you have to have an account in order to add textbooks. With your account, you can look through your own personal listings from your dashboard to edit or delete items. The add, edit and delete pages are all password protected, and they also verify that the user logged in is also the owner of the textbook.

My two added features are the file upload system where users upload a photo of their textbook, and different user permission levels (guests can only search, registered users can add and edit textbooks).

About the Database

The database I am using is pictured above. Anytime someone registers with a new account, that information will be stored in the users table. Any class prefixes for USC will be manually updated in the classcode table. When someone adds a new textbook entry, their user ID will be a foreign key to the users table, and the classcode int will be a foreign key to the classcode table.

All information besides the class codes will come from user input when they register their account and add textbooks.
